Instructions for use

1. Total amount to be claimed is displayed automatically.

2. Tap + to add a new receipt

Select type of claim.

Insert location of claim.

Insert amount to be claimed from this receipt.

Scan receipt, or rescan if required.

Tap log expense.

All fields are required.

3. Tap a claim to see complete details.

Tap "Delete Expense" if required.

Known issues: Data does not save to device storage. If the app is closed all expenses will be lost.
